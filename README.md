# translating

### 文献翻译小助手
1. 实现将文章分句翻译,去除从pdf粘贴来的多余换行。
2. 实现划词翻译，选中英文单词或句子即可实时翻译
3. 自动储存翻译状态，关闭网页后重新打开，所翻译的内容不会丢失
